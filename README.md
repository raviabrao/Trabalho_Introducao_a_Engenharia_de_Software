# Fábrica de Campeões

> Plataforma mobile para controle de treinos e acompanhamento de atletas.

---

## Sobre o projeto

A **Fábrica de Campeões** é um aplicativo mobile criado para ajudar atletas e técnicos no gerenciamento de treinos, desempenho e comunicação. O sistema funciona em Android e iOS.

Projeto desenvolvido para a disciplina de Engenharia de Software — 2026.

---

## Funcionalidades

- Cadastro e login de usuários
- Registro de treinos
- Histórico de desempenho
- Criação de planos de treino
- Feedback do técnico
- Agendamento de competições
- Relatórios automáticos
- Login com Google e Apple

---

## Atores do sistema

| Ator | Função |
|---|---|
| Atleta | Registra treinos e acompanha desempenho |
| Técnico | Cria planos e acompanha atletas |
| Sistema | Gera relatórios e envia notificações |

---

## Fluxo principal

1. Usuário faz login
2. Registra um treino
3. Sistema valida os dados
4. Informações são salvas
5. Relatório é gerado
6. Confirmação enviada ao usuário
